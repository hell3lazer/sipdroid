# Sipdroid (Modernized Fork)

Sipdroid is an open-source SIP client for Android. This fork modernizes the original Sipdroid application with updated user interfaces, dynamic theme support, and quality-of-life improvements while retaining its robust and historic SIP core.

## 🚀 Features & Changes in this Fork

- **Material Design 3 Upgrade:** The application UI has been fully revamped using Material Design 3 principles, including updated padding, sizing, and iconography.
- **Dynamic Theme Support:** Full support for system-level Light and Dark themes, seamlessly adapting backgrounds and text colors.
- **Redesigned Home Screen:** Cleaned up the home screen by removing outdated buttons, standardizing to square buttons, centralizing the dialpad button, and implementing custom corner sizing.
- **Modernized In-Call Experience:** 
  - The in-call dialpad now correctly wraps its height, snapping elegantly above the call controls.
  - The DTMF text input field has been transformed into a sleek, transparent native input.
  - The call card profiles have been streamlined, hiding unused elements to maximize screen real estate and prevent clipping.
- **Cleaner Notifications:** Ongoing call notifications now display a clean, readable Caller ID rather than leaking the raw SIP connection URIs.
